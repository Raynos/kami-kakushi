# brainstorms/

Raw discovery / Q&A capture — the working notes from grilling out a design, **before** it's settled.
The [`grill-me`](../.claude/skills/grill-me/SKILL.md) skill writes here: one file per session,
`YYYY-MM-DD-{topic-slug}.md`.

These are *not* the source of truth. Once a design settles, it graduates to [`../docs/`](../docs)
(and decisions to [`../human-in-the-loop/decisions.md`](../human-in-the-loop/decisions.md) or a
decision log). Brainstorms stay as the historical record of *how* the thinking went.
