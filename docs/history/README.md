# docs/history/

How the design **got here** — the opposite of `docs/` (which is always *current truth*). Nothing
here should be read as "the current state of the game".

- **[decisions.md](decisions.md)** — the **ADR log** (`D-000…`): every locked decision, its options,
  and *why* one was chosen — including superseded ones. The authoritative record of *rationale*.

> When a living doc (`docs/…`) and an ADR disagree on *current state*, the living doc wins (fix it if
> it's stale). On *why* a thing is the way it is, the ADR log wins.

Per-session work narrative lives in [`../../journal/`](../../journal/).
