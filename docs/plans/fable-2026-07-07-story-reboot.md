## §1 · The frame (human-locked, 2026-07-07)

Four decisions, each answered explicitly in-session:

1. **Scope: the design layer is REOPENED — not just the prose surface.**
   The audit's "architecture is good, rewrite the words" split is NOT
   adopted as a boundary. Cast, arcs, tier story, premise framing, the
   Tama reveal, PRD §5 end to end: all of it is input that re-earns its
   place, none of it is canon by default.
2. **Fixed ground: the kernel ONLY.** The seven-point spirit distillation
   (lie of the vanishing · nobody twice unnamed · worth written by work ·
   house before the man · partial justice · form is the fiction · woodblock
   restraint) is the only binding seed — and even it gets a redline pass
   first, since it was distilled FROM the old story and now carries
   everything.
3. **Mechanics: FULL CO-DESIGN.** The story pass may reshape tier/rung
   meaning, promotion structure, node design, progression feel — story and
   mechanics are derived together, studio-style. Mechanical changes still
   land as HD-items/ADRs when they cut canon.
4. **In-flight builds RUN in parallel.** emergent-node-actions (and any
   other live workstream) continues untouched; whatever it ships becomes
   concrete input the co-design keeps, bends, or supersedes on merit. No
   repo freeze, no coordination interrupt. The current T0 game keeps
   shipping as-is until the reboot's build wave replaces it.

**Process shape: pillars → pitches → co-write.** Divergence where
taste-risk is highest (the macro pitch), ping-pong everywhere else.

## §3 · The phases

- **Phase 0 — frame.** ✅ This section; locked 2026-07-07.
- **Phase 1 — pillars co-write (in-chat, ping-pong, no subagents).**
  1. Kernel redline — re-affirm/amend the seven points with fresh eyes.
  2. Fiction pillars — what the story must be/never be (audit anti-goals
     distilled; register rules as candidates).
  3. Mechanics-feel pillars — what progression/session/friction must FEEL
     like, since mechanics are co-designed.
  4. Constraints sheet — platform, genre (incremental), scope realities
     (T0 first), the ADR-139/scorecard machinery that still applies.
  Output: the pillars section, committed — the constitution every pitch
  must honor.
- **Phase 2 — pitches (macro diverge, MIXED FLEET of 5).** Five
  independent agents, blind to each other, each authoring a COMPLETE story
  pitch: premise, mystery spine, core cast, tier/act arc, and how
  mechanics and fiction cause each other. **3 blind** (kernel+pillars
  only — purest divergence) + **2 informed** (also get PRD §5, the built
  narrative, and the audit, instructed "input, not canon") — blind/informed
  convergence in the bundle is signal, not anchoring. Reviewed as one
  coherent bundle (HR-item); the human picks/blends.
- **Phase 3 — co-write the design.** From the winning pitch: the story
  bible (cast, tiers, world texture, register rules) drafted section by
  section, redlined live, committed when locked. Mechanics decisions
  surface as HD/ADR as they land.
- **Phase 4 — re-plan the build.** From the locked bible: the prose wave
  (ADR-139 bundles), the systems build plans, the PRD §5 rewrite/ripple,
  the migration plan for built T0 content. Each sized as its own plan.
