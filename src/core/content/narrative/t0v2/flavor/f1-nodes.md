# F1 — the node sheet · T0 zone flavor

**Contract.** Single law-compliant take, no rivals (human ruling: the
flavor layer ships one version). Source spec: the zones section of
[`docs/story-bible/tiers/t0.md`](../../../../../docs/story-bible/tiers/t0.md);
prose bound by the kernel + §0.5. Narrator voice throughout — plainest
register, second person for the MC, **no epigrams** (the license is
Munemasa's and Kihei's, and neither speaks on this sheet).

**Keys, per zone (mechanical migration):**

- `id` — stable node key (heading backtick).
- **Display** — the map label the player reads (given only where it
  differs from the spec heading).
- **Blurb** — the node's current-location flavour (taste #15: terse, no
  next-zone hints, no destination previews).
- **Wrong thing** — the surfaced fiction line(s), carrying the seeding
  exactly as t0.md places it. *Staging notes in square brackets are
  NOT fiction voice* — gates, frequency, and pointers to the authored
  units that own the adjacent prose.
- **Actions** — `action-key` — "Label". Labels are fiction-voiced
  buttons: period verbs, no abstract game-words.
