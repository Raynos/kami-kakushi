Kami-kakushi — UI-remaster variants

Five full remasters of the actual game's T0 (R0–R3) UI — same tabs, same verbs, same numbers, same log-centric idle RPG — each a generation better in its own direction. The midpoint between "change the CSS" and "reinvent the UI."

Each runs on a shared mock engine: pick a stage (cold open → R3) in the bottom-right strip, then play — rake, walk, trade, face the wolf, answer the summons. English UI throughout; golden-age JRPG spirit, no cosplay.
