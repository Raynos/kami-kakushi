Kami-kakushi — UI-remaster variants

Six full remasters of the actual game's T0 (R0–R3) UI — same tabs, same verbs, same numbers, same log-centric idle RPG — each a generation better in its own direction, plus a three-variant Moonlit × Lacquer fusion round (07–10). The midpoint between "change the CSS" and "reinvent the UI."

✅ Locked (R9, 2026-07-04): 10 Andon Steel is UI-v2's direction (ADR D-127). 01–09 remain below as the exploration record the pick was made against — nothing here ships; the port into the real game is a future build.

Each runs on a shared mock engine: pick a stage (cold open → R3) in the bottom-right strip, then play — rake, walk, trade, face the wolf, answer the summons. English UI throughout; golden-age JRPG spirit, no cosplay.
