# archive/

Where **superseded or finalized markdown** goes to rest — design docs and plans no longer active but
worth keeping for historical reference. **Markdown only**; not code, not build artifacts.

- **Archive** when a doc is superseded by a newer one, or a plan is fully done/obsolete — move it
  here instead of deleting, so the reasoning is still recoverable.
- **Delete** only when an idea is truly discarded with no future reference value.

The *why* of a move belongs in a journal entry (and, for decisions, the ADR log).
