# T0+T1+T2 map blind-pass — ensemble report (2026-07-09)

- **Capture dir:** `project/audit/screens/2026-07-09-map-blind-pass`
- **Sheets covered:** T0, T1, T2 — this is a **scoped run**, not a full pass
  (T3+ sheets, if any exist, are out of scope here).
- **Ensemble design:** 3 independent blind readers per sheet. Each reader was
  shown the sheet cold (no line list, no prior answer) and wrote a full
  from-scratch description. Each reader's description was then judged,
  independently, against every line in that sheet's spec (M = must-recover /
  major line, S = secondary / scenic line). Per-line verdict = **strict
  majority** across the 3 judged reads (2-of-3 or 3-of-3 pass = PASS overall;
  0-of-3 or 1-of-3 pass = FAIL overall). A single-reader run is a **sample**,
  never a verdict — the ensemble is what carries weight here.
- **Overall verdict: FAIL.** Multiple must-recover (M) lines fail majority on
  T1 (the tier with the most content), and every tier drops at least one
  secondary (S) line to 0/3 or 1/3.
