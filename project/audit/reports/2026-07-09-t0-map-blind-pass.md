# T0 map blind-pass — ensemble report (2026-07-09)

**Capture dir:** `project/audit/screens/2026-07-09-map-blind-pass`
**Sheets covered:** T0 only — this is a SCOPED run, NOT a full map-sheets
pass (T1/T2 are not evaluated here; see other dated reports for those).

**Ensemble design:** 3 independent blind readers per sheet. Each reader
saw the capture cold, with no access to the other readers' output, and
wrote a full freehand description before any rubric was applied. Each
line of the map-spec §5 rubric is then scored per reader (pass/fail),
and the per-line verdict is the **strict majority** of the 3 votes
(2-of-3 or 3-of-3 to pass). A single-reader run is a SAMPLE, never a
verdict — the majority-of-3 design exists specifically so one reader's
blind spot can't sink (or save) a line, and so a unanimous miss reads
as a real, reproducible gap rather than one reader's idiosyncrasy.

**Overall verdict: FAIL**

it clears the majority bar, because two different readers land on two
different reads of the same ink — that's not settled, it's lucky.
