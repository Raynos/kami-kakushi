# T0+T1 map ensemble blind-pass — 2026-07-09

**Capture dir:** `project/audit/screens/2026-07-09-map-blind-pass`
**Sheets covered:** T0, T1 (a scoped run — this is NOT a full pass covering
every tier sheet in the game).

**Ensemble design:** 3 independent blind readers per sheet. Each reader is
given only the rendered sheet (no roster, no prior descriptions) and writes a
full free-form description with no knowledge of the other readers' output or
of the roster being graded against. Each reader's description is THEN judged,
separately, line-by-line against the roster. Per-line verdict is **strict
majority** across the 3 judgings (2-of-3 or 3-of-3 pass = line passes;
2-of-3 or 3-of-3 miss = line fails). A single-reader run is a **SAMPLE**,
never a verdict — the whole point of the ensemble is that one reader's blind
spot doesn't sink a line, and one reader's lucky guess doesn't save one
either.

**Overall verdict: PASS**

Both sheets clear on majority-line and clear-through-repetition (M-kind)
lines at 100%. The only failures are S-kind (subtler/secondary) lines, and
they fail for the *same* underlying miss in both tiers — the orchard's
courtyard-row alignment to the ruin, and its bamboo-grove pressure on the
vegetable rows — which reads as one recurring soft spot in the visual
composition rather than two unrelated defects.
