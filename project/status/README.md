# memory/

Durable, important facts about **this repository** — one `.md` file per fact, committed and
reviewable. For things easy to lose and costly to rediscover: conventions to follow next session,
non-obvious rationale, external reference links.

Distinct from `docs/` (living design), `journal/` (per-session logs).
