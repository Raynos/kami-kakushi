# Reference screenshots — the two README inspiration games

Captured by the human 2026-07-03 (D-126: the taste top-layer lock names
these as the density/chrome references; README.md L26–30 has the links).
These are the **evidence** behind taste.md's two reference lines — the
lines stay one-liners, the detail lives here.

## proto23 — <https://23html.github.io/>

Boxed multi-panel dashboard; every panel framed, dense, tiny type; one
semantically colour-coded message log carries the play.

- `01-training-grounds-fresh-start.png` — lvl-1 start: You-panel (hp/exp/
  energy bars, stat row, equipment grid with **LOCKED slots visible**),
  enemy panel, tabbed inventory, location card with in-fiction calendar +
  season glyph (春), the colour-coded log (items/EXP/misses each their
  own colour).
- `02-dojo-lobby-item-shower.png` — mid-progress: title earned ("Glass
  Bones"), item-shower moment in the log — a dozen "New item obtained"
  lines each colour-tagged; inventory filling up.
- `03-home-node-discoverable-actions.png` — the Home node: location
  flavour text + **discoverable in-fiction action rows** ("Crash down and
  take a nap" / "Examine basement door" / "Examine Fireplace" / "Go
  outside") — the emergent-node-actions shape, live.

## yet-another-idle-rpg — <https://miktaew.github.io/yet-another-idle-rpg/>

Three-column IA (vitals+stats | location/activity | skills+quests+log);
many small visible progressions; idle math stated in plain numbers.

- `01-infested-field-combat.png` — combat at a location: enemy stat strip
  + hp bar in the location column, **per-skill progress bars** (level/cap
  + %) grouped Activity/Combat/Environmental, combat log far right,
  in-fiction datetime + weather strip.
- `02-village-elder-dialogue.png` — NPC dialogue inline in the location
  column: described in prose (no portrait), question options as rows,
  explicit "Go back".
- `03-village-fieldwork-eta.png` — an activity card that shows its
  arithmetic: "1 base xp per in-game minute to Farming (44.5% [72/164]),
  next level in 1 hour (2 minutes realtime), next earnings in 1h57m" —
  **rate + ETA legibility**.
